# Contributing

First, Thank you!

Your contribution is very welcomed!

To contribute, either open a PR or request committer access.

# Conventions

## Package naming

Use:
- Lowercase
- "-" as name separator
- "_" as version separator
Example: `package-name_1_2_3`

- Use upstream package name. Indexes:
  - Python: `pypi.org`
  - OCaml: `opam.ocaml.org`
